# Курсовая работа: Веб-приложение "Ювелирная Мастерская ООО Аваль"

Данный проект представляет собой курсовую работу по теме "Разработка интернет-магазина для ювелирной мастерской". Веб-приложение создано на фреймворке Laravel и представляет собой полнофункциональную e-commerce платформу, включающую в себя публичную часть для клиентов и административную панель для управления контентом и заказами.
